#Color changes

**Color Changes** is a simple implementation of color changes by using loop. There are three different shapes in this implementation, they are stripes, pie, and snowflakes.

## Usage
1. Clone the repository above
2. Open the project's files using any IDE (Eclipse preferrably)
3. Import external library (uwcse.jar)
4. Run the module program
